Однорукий бандит (Slot Machine)

Курсовая работа по программированию на C#.

Описание
Игровой автомат с тремя барабанами на Windows Forms.

Технологии
- .NET 8.0
- C#
- Windows Forms
- MSTest

Как играть
1. Выбрать ставку (5-50)
2. Нажать "Крутить"
3. Три одинаковых символа = выигрыш!

Запуск
Открыть `SlotMachine.sln` в Visual Studio → F5
