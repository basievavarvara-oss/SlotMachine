# 🎰 Однорукий бандит (Slot Machine)

Курсовая работа по дисциплине «Программирование».  
Тема: «Разработка игры "Однорукий бандит" на языке C#».

---

## 📋 Описание проекта

Данное приложение представляет собой классический игровой автомат с тремя барабанами.  
Игрок делает ставку, запускает вращение и получает выигрыш при выпадении трёх одинаковых символов.

**Цель игры:** набрать как можно больше фишек, делая ставки и выигрывая на удачных комбинациях.

## 🎮 Правила игры

1. Начальный баланс составляет **100 фишек**
2. Игрок выбирает ставку от **5 до 50** (шаг 5)
3. При нажатии кнопки **«Крутить»** ставка списывается с баланса
4. Три барабана останавливаются в случайных позициях
5. Если выпало **три одинаковых символа** — игрок получает выигрыш
6. Если символы не совпали — выигрыша нет

## 🚀 Запуск приложения

1. Открыть файл `SlotMachine.sln` в Visual Studio 2022
2. Убедиться, что проект `SlotMachine.UI` назначен запускаемым
3. Нажать **F5** или кнопку «Пуск»

---

## 🧪 Запуск тестов

1. Открыть проект в Visual Studio 2022
2. В верхнем меню выбрать: **Тест** → **Обозреватель тестов**
3. Нажать кнопку **«Запустить все тесты»**
4. Все 4 теста должны отобразиться с зелёными галочками
